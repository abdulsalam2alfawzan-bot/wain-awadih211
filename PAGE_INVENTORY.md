# قائمة الصفحات والوظائف

| الصفحة | الوظيفة |
|---|---|
| `index.html` | الصفحة الرئيسية والوعد الرئيسي وبدء الطلب والمتابعة |
| `customer.html` | الموقع، بيانات العميل، OTP، السيارة، وصف المشكلة، الأسئلة، التوجيه، الموافقة وإنشاء الطلب |
| `track.html` | استعادة الطلب ومتابعته وطلب بديل والتقييم |
| `workshop-accept.html` | رابط الورشة الخاص لمراجعة الإحالة وقبولها أو رفضها |
| `join.html` | طلب انضمام الورشة وبيانات Google Place ID |
| `join-status.html` | متابعة طلب انضمام الورشة |
| `admin-login.html` | دخول فريق الإدارة الداخلي |
| `admin-dashboard.html` | إدارة الطلبات والإحالات والشركاء وربط Google والسجلات المالية والإدارية |
| `privacy.html` | سياسة الخصوصية والموقع والتحقق ومصادر التقييم |
| `terms.html` | الشروط وحدود التوجيه والإحالة |
| `404.html` | معالجة الروابط غير الموجودة |
| `workshop-login.html` | صفحة محفوظة وغير متاحة حاليًا |
| `workshop-dashboard.html` | صفحة محفوظة وغير متاحة حاليًا |
| `payment.html` | صفحة محفوظة وغير متاحة حاليًا |
| `receipt.html` | صفحة محفوظة وغير متاحة حاليًا |
